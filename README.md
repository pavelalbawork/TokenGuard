# TokenGuard

**TokenGuard** is a niche macOS menu bar utility designed for developers to track their AI tool usage and subscriptions directly from their local environment. 

Rather than requiring API keys or cloud syncing, TokenGuard reads the local state files generated by the CLIs and apps you already use (like Claude Code or Codex). It brings your scattered AI usage data into a single, quickly accessible menu bar popover.

*Note: TokenGuard is an independent developer utility and is not affiliated with, endorsed by, or sponsored by OpenAI, Anthropic, or any other AI service provider.*

## Who is this for?
Developers and power users who rely on multiple AI tools and want a lightweight, local-first way to keep track of their active sessions, remaining request quotas, and subscription statuses without leaving their workflow.

## Why does it exist?
Keeping track of quotas across different AI tools often means opening multiple web dashboards or running various CLI commands. TokenGuard aggregates this data by surfacing what your machine already knows, giving you a unified view without requiring separate API keys or cloud authentication.

## Supported Providers

## Installation

TokenGuard is distributed directly via GitHub Releases. It is a signed and notarized macOS application.

1. Go to the [Releases page](https://github.com/pavelalbawork/TokenGuard/releases).
2. Download the latest `TokenGuard.zip` file.
3. Unzip the file and drag `TokenGuard.app` to your `Applications` folder.
4. Launch the app. It will appear in your macOS menu bar.

## Setup & Prerequisites

Because TokenGuard relies on local provider state, **you must have the corresponding provider's CLI or application installed and authenticated on your machine.**

- If you want to track Claude, ensure you have authenticated via the `claude` CLI locally.
- If you want to track Codex, ensure its local environment is set up and logged in.

*TokenGuard does not log you in to these services; it only reads the state they leave behind.*

## Privacy & Trust

**Your data stays on your machine.** 
TokenGuard works by reading local configuration and session files generated by your existing CLI tools. 
- It does **not** send your usage data, session tokens, or personal information to any third-party servers or analytics platforms.
- It does **not** require your API keys.

## Limitations

- **Fragility to Upstream Changes**: Because this app parses local files and undocumented local CLI states, it will break occasionally if Anthropic, OpenAI, or other providers change their local data formats. We update parsers as needed.
- **Accuracy**: Data is only as fresh as your local state. If you use an AI tool on another device, this utility won't know about it until your local CLI syncs.
- **macOS Only**: Currently built specifically for the macOS menu bar.
